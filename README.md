# taifatang.cv

Curriculum Vitae written in [Typst](https://typst.app). For a full career timeline, portfolio, and project highlights, visit [taifatang.github.io](https://taifatang.github.io).

## Usage

Compile to PDF:
```sh
typst compile cv.typ
```

## Deployment

Any changes to the main branch will trigger a deployment to [taifatang.github.io](https://taifatang.github.io) via GitHub Action
